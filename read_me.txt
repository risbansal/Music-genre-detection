READ ME

Language: Python3

Libraries:
pandas
numpy
nltk
scipy
sklearn


File decription:

Music_analyzer.py - To train and test audio data for genre classification using 5 classifiers
Sentiment_extractor.py - To extract sentiments and train on 5 classifiers

features.csv - dataset of audio features with genre labels
features2.csv - copy of features.csv - used for classification with sentiments
sentiment.csv - song information for sentiment extraction

Running the code:
1. Keep all .py and .csv files in one folder
2. Open terminal with current directory as the folder containing codes and datasets
3. Run code with python3 file_name.py(python3 Music_analyzer.py OR python3 Sentiment_extractor.py)
